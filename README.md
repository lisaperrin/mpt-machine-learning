# Portfolio Optimization with Machine Learning

A finance project implementing Modern Portfolio Theory with machine learning across multiple asset classes for optimal diversification and risk-adjusted returns.

## Overview

This project implements advanced portfolio optimization techniques using:
- **Modern Portfolio Theory (MPT)** - Classic Markowitz optimization
- **Machine Learning Enhanced Models** - Random Forest predictions with technical indicators

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run analysis
python main.py
```

## Performance Metrics

The system evaluates portfolios using:
- Annualized returns and volatility
- Sharpe ratio and information ratio
- Maximum drawdown
- Value at Risk (VaR)
